Real-Time Chat Application

A full-stack real-time chat application built with Spring Boot, MongoDB, WebSocket (STOMP + SockJS), React, and Tailwind CSS.

Features:
Create and join chat rooms, 
Real-time messaging using WebSockets, 
Persistent chat history in MongoDB, 
Responsive React UI

Tech Stack:
Backend:
Java 21, 
Spring Boot, 
Spring WebSocket, 
MongoDB, 
Maven, 

Frontend:
React, 
Vite, 
Tailwind CSS, 
STOMP.js, 
SockJS,
